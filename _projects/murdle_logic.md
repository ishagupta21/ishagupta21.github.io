**[Play Murdle Logic](https://zippy-kitsune-f0310b.netlify.app){: target="_blank" rel="noopener noreferrer"}** — no install required.

Murdle Logic is a lightweight web app for **grid-style deduction puzzles**: you use clues to eliminate possibilities, mark the grid, and reach a single consistent solution. It runs entirely in the browser.

I built this one **Saturday afternoon** when I was bored and wanted something small and playful to ship.
